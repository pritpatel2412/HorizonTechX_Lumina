# Project Summary
This project is a web application designed for lifestyle creators to share daily inspirations related to coffee, travel, and positive vibes. It enables users to create and showcase personalized profile pages, enhancing their online presence and fostering connections with followers. The app features a vibrant interface for content sharing, monetization options for exclusive content, and improved mobile responsiveness. Recent updates include a light pink theme, a "Load More Supermodel" button for dynamic content loading, and personalized model names for each supermodel post. The platform now supports video content for paid cards, enhancing user engagement and monetization opportunities.

# Project Module Description
The project consists of the following functional modules:
- **Profile Management**: Manages creator profiles, including personal information, posts, and social links.
- **Content Display**: Renders the creator's content in a feed-style layout with interactive elements.
- **Interactivity**: Features follow buttons, social media links, and post interactions (likes and comments).
- **Monetization**: Provides sections for paid content and subscription options to enhance revenue streams, now with video support for paid cards.
- **Video Integration**: Supports video posts with automatic play and seamless transitions, now including a poster image to avoid loading issues.
- **Dynamic Content Loading**: Implements a "Load More Supermodel" button to dynamically reveal additional media posts.

# File Description Inventory
- **README.md**: Overview of the project and setup instructions.
- **src/data/profile.ts**: Contains static data for profiles, including social links, posts, monetization content, video post metadata, and personalized model names.
- **src/pages/Index.tsx**: Main page component rendering the creator profile and content in a feed-style layout, now includes a "Load More Supermodel" button and personalized names for each supermodel post.
- **public/assets/profile/avatar-new.jpg**: Updated user-uploaded profile avatar.
- **public/assets/profile/cover.jpeg**: Cover image for the creator profile.
- **public/assets/models/**: Directory containing new uploaded model images and video assets for posts.
- **public/assets/cards/**: Directory containing video assets for paid cards.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: Superset of JavaScript that adds static types.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool providing fast development and hot module replacement.
- **PostCSS**: Tool for transforming CSS with JavaScript plugins.

# Usage
1. Install dependencies:
   ```
   cd app/frontend
   pnpm install
   ```
2. Lint the code:
   ```
   pnpm run lint
   ```
3. Build the project:
   ```
   pnpm run build
   ```
